# odin-recipes

## Overview
A simple recipe web app built using just HTML. This project showcases a collection of recipes with detailed descriptions, ingredients, and preparation steps.

## Features
- **Homepage**: Lists all available recipes with links to their individual pages.
- **Recipe Pages**: Each recipe has its own dedicated page with:
  - A brief description of the dish.
  - A list of ingredients categorized for easy understanding.
  - Step-by-step preparation instructions.
  - Images of the dishes for visual appeal.

## Recipes Included
1. **Lasagna**
   - A classic Italian dish with layers of pasta, meat sauce, and cheese.
2. **Chicken Fried Rice**
   - A flavorful stir-fried rice dish with chicken, vegetables, and savory sauces.
3. **Margherita Pizza**
   - A simple yet delicious pizza with fresh tomato sauce, mozzarella cheese, and basil leaves.

## How to Use
1. Open the `index.html` file in any web browser.
2. Click on the recipe name to navigate to its detailed page.

## File Structure
- `index.html`: The homepage listing all recipes.
- `recipes/`: Contains individual HTML files for each recipe.
  - `lasanga.html`: Details for Lasagna.
  - `chicken_fried_rice.html`: Details for Chicken Fried Rice.
  - `Marghreita_pizza.html`: Details for Margherita Pizza.
- `images/`: Contains images used in the recipe pages.
